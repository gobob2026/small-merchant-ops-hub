# client

## Stack
- Nuxt 4

## Run
```bash
npm install
npm run dev
```

## Runtime Config
- `NUXT_PUBLIC_API_BASE` (default: `http://127.0.0.1:8080`)

## Available Flow
- Create member (name/phone/channel)
- Create order (member/amount/source/status)
- View real-time summary (member count, revenue, repurchase rate)
